// Projects

{ Fiber } { F1 Line } { Nebula Store }
Progetto realizzato per l'Istituto Tecnico Superiore Kennedy di Pordenone su un prototipo di scatola di cereali, io mi sono occupato di realizzare il prototipo della pagina web del prodotto.
Probabilmente il progetto più grande che io abbia mai fatto. Cominciato nell'estate del 2014 partendo da un blocco note, sino ad arrivare ad un sito web con una modesta comunità di giocatori appassionati di Formula 1. Io e il mio socio ora gestiamo uno dei migliori tornei online per PC riguardante i capitoli del gioco F1 by Codemaster. Il sito non è attualmente responsivo, ma verrà presto aggiornato.
Questo progetto, sempre realizzato per conto dell'ITS Kennedy di Pordenone, prevedeva di mostrare la lista negozi ed il relativo dettaglio della società Jesse. Io ho curato la parte front-end del progetto.
